# Disneyland Food

## Overview

Provide general, non-time-sensitive guidance on Disneyland dining, including types of dining, how to choose, and dietary considerations.

## Quick Start

1. Ask for constraints: party size, budget sensitivity, dietary needs, time window, and vibe (quick snack vs sit-down).
2. Explain dining types and tradeoffs (quick-service vs table-service vs snacks).
3. Offer a short, tailored suggestion list framed by the user's constraints.
4. If the user needs current menus, prices, hours, or closures, ask if you should look them up.

## Task-Based Guidance

### Clarify the Request

Ask 2-4 of these as needed:
- Visit timing (morning, midday, evening)
- Party makeup (kids, adults, picky eaters)
- Dietary needs (vegetarian, vegan, gluten-free, allergies)
- Desired pace (grab-and-go vs relaxed meal)
- Budget sensitivity (low, medium, splurge)

### Explain Dining Types

- Quick-service: faster, less planning, good for flexibility.
- Table-service: slower, reservations often needed, better for a long break.
- Snacks/stands: best for grazing or short waits between attractions.

### Recommend Without Time-Sensitive Claims

- Provide category-based suggestions rather than specific menu items.
- Emphasize balancing convenience, taste preferences, and time.
- Offer 2-4 options per category to keep it actionable.

### Dietary and Accessibility Guidance

- Note that most venues can accommodate common dietary needs, but details vary.
- Encourage the user to check current allergen and ingredient info if needed.
