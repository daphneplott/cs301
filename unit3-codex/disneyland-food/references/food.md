# Disneyland Food Reference

## Use This When

- The user asks for recommendations but does not need current menus, prices, or hours.
- The user wants help choosing between dining types or planning food breaks.

## Intake Questions (Pick 2-4)

- What time of day are you eating?
- Any dietary needs or allergies?
- Do you want quick bites or a sit-down meal?
- Are you trying to minimize time away from attractions?
- How picky is your group (kids, adventurous eaters)?

## Recommendation Heuristics

- If the user wants speed: prioritize quick-service or snack stands.
- If the user wants a longer break: suggest table-service and mention reservations.
- If the user is grazing: suggest mixing a couple of snacks with one quick meal.
- If the user is budget-sensitive: favor quick-service and snack combinations.
- If the user needs dietary options: emphasize that offerings vary by location and to confirm ingredient details.

## Language Templates

- "If you want something fast, quick-service spots are usually best. If you want a longer break, table-service is more relaxing but may need reservations."
- "Do you have any dietary needs or a preferred vibe (quick snack vs sit-down)?"
- "I can suggest categories and a plan, and if you need current menus or prices I can look those up."

## Avoid

- Stating specific menu items, prices, or hours as facts unless the user provides them.
- Claiming availability of seasonal or limited-time offerings.
- Overly detailed location-by-location lists that become time-sensitive.
