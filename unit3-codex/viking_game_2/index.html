Viking Game 2

Hnefatafl 11x11 — Hotseat, click-select, minimalist

Turn: Attackers
Captured (A/D): 0 / 0
Message: Select a piece to begin.

Quick Rules

  • Attackers (black) vs Defenders (white) + King (gold).
  • Move any number of squares orthogonally, no jumping.
  • Only the King may enter the throne (center) and corners.
  • Capture by sandwiching between two opposing pieces.
  • King wins by reaching a corner; attackers win by 4-side capture.
