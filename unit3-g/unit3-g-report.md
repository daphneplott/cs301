It seems like the openai has a good system for evaluation, although you can’t quite run it the same way as a normal python program. When you create the eval, you have to tell it what data input to expect, and what criteria to grade by. You have to keep track of the specific ID given to the eval object. You then upload your test data, and have to keep track of the id that is given. Then you can create an evaluation run by giving it those 2 ids, as well as information for which model and prompt to use. It’s interesting that it is all done through curl http requests, instead of having a python code api.
